hexo+github简单命令

Feb 10, 2018

简单命令

创建文章、标签等

  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面

修改文章发布、同步到github等

  • hexo p == hexo publish
  • hexo g == hexo generate #生成
  • hexo s == hexo server #启动服务预览
  • hexo d == hexo deploy # 部署到github
